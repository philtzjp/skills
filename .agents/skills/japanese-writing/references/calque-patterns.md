# 直訳パターン対照表

英語（口語・スラング・技術用語）から来やすい不自然な日本語と、その自然な言い換え。

## 動詞・スラング系

| 英語のニュアンス | 直訳調（避ける） | 自然な日本語 |
|---|---|---|
| land (a PR / changes) | 寄せる、着地する | マージする、取り込む、反映する |
| drop (feature / support) | 落とす、ドロップする | 削除する、廃止する、提供を終了する |
| knock down / take down | 倒す | 修正する、解消する、取り除く |
| ship / push out | 出す、送り出す（文脈不明） | リリースする、公開する |
| roll out | 転がす、ロールアウトする | 段階的に展開する、順次提供する |
| spin up | 回す、スピンアップする | 起動する、プロビジョニングする |
| wire up | 配線する | 接続する、組み込む |
| bake in | 焼き込む | 標準装備にする、組み込む |
| bubble up | 浮上する（乱用） | 表面化する、上位に伝わる |
| nail down | 釘付けにする | 確定する、詰める |
| call out | 呼び出す | 明示する、指摘する |
| follow up | フォローアップする（乱用） | 追って連絡する、後日対応する |
| circle back | 円を描いて戻る | 改めて検討する、後ほど戻る |
| unblock | アンブロックする | ブロックを解除する、障害を取り除く |
| green-light | 青信号を出す | 承認する、着手を許可する |

## 接尾語・構文系

| パターン | 問題 | 対処 |
|---|---|---|
| 〜きる（書ききる、やりきる、読みきる） | 英語の "finish ~ing" / "~ out" の直訳になりやすい | 〜し終える、〜を完了する、〜まで行う |
| 〜していく | 英語の "going to ~" の直訳調 | 〜します、今後〜します |
| 〜することができる | 冗長 | 〜できる |
| 〜する必要がある | 硬い・直訳調 | 〜が必要です / 〜してください |
| 〜に関して | 硬い | 〜について / 〜に関する（名詞修飾） |
| 〜の方 | 冗長 | 〜（単独で十分なら削る） |
| 〜となります | 直訳調の断定 | 〜です / 〜になります（必要な場合のみ） |

## 文末・依頼系

| 直訳調（避ける） | 自然な日本語 |
|---|---|
| 〜どうぞ。 | 〜ください。 / 〜をお願いします。 |
| ご確認どうぞ。 | ご確認ください。 |
| お試しどうぞ。 | お試しください。 |
| 〜まで。 | 〜までです。 / 〜予定です。 / 〜を予定しています。 |
| リリースまで。 | リリース予定です。 |
| 対応まで。 | 対応中です。 / 対応予定です。 |
| 以上。 | 以上です。 / 以上となります。（文脈次第） |
| よろしく。 | よろしくお願いします。（ビジネス文脈） |

## 体言止め（名詞＋句点）

本文では次のような体言止めを避け、述語を補う。

| 体言止め | 修正案 |
|---|---|
| 対応。 | 対応しました。 / 対応します。 |
| 修正。 | 修正しました。 / 修正が必要です。 |
| 確認。 | 確認してください。 / 確認済みです。 |
| デプロイ。 | デプロイしました。 / デプロイします。 |
| 未対応。 | 未対応です。 / まだ対応していません。 |
| 要確認。 | 確認が必要です。 |

**例外:** UI のステータスラベル（「対応中」「未対応」）、見出し、極めて短い箇条書き。

## カタカナ語

カタカナ語は悪ではないが、日本語で十分伝わる場合は和語を優先する。

| 過剰なカタカナ | 和語・一般語 |
|---|---|
| インプリメンテーション | 実装 |
| デプロイメント | デプロイ / 展開 |
| レファレンス | 参照 / 参考 |
| コンフィグレーション | 設定 |
| バリデーション | 検証 / 入力チェック |
| リファクタリング | リファクタリング（定着語は可） / 整理 |

## 判断に迷ったとき

1. その文を日本語のネイティブが日常・業務でそのまま言うか想像する
2. 英語原文を頭の中で復元し、直訳になっていないか確認する
3. 述語が欠けていれば補う
4. 同じ語尾（〜します、〜できます）が 3 回以上続く場合は文を変える
